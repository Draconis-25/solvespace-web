SolveSpace's website rework
====================

This repository contains the source code of the official website of [SolveSpace], a parametric
2d/3d CAD.

[solvespace]: https://solvespace.com

Prerequisites
-------------
- Python 3.7+
- [Bottle](https://bottlepy.org/docs/dev/).

### Running
```sh
python main.py
```
